# To-Do Seznam

Jednoduchá webová aplikace pro správu úkolů. Uživatel může přidávat úkoly, označovat je jako splněné a mazat je. Všechny úkoly se ukládají do `localStorage`, aby zůstavaly zachovány i po obnovení stránky.

## Použité technologie

* HTML5
* CSS3
* JavaScript (ES6)
* localStorage

## Funkce

* Přidávání nových úkolů s časovým razítkem
* Označení úkolu jako hotového kliknutím
* Odstranění úkolu
* Trvalé uložení úkolů v `localStorage`

## Jak spustit

1. Stáhni nebo naklonuj repozitář:

   git clone https://github.com/vaclavprucha/JavaScriptProjects.git
   
2. Otevři soubor `index.html` v prohlížeči.
